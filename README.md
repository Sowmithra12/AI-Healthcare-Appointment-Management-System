# AI-Powered Healthcare Appointment Management System

## Features

- Appointment Booking Agent
- Queue Monitoring Agent
- Doctor Recommendation Agent (Ollama + Qwen 3 4B)
- Appointment Cancellation Agent
- Intelligent Rescheduling Agent
- Email Reminder Agent
- Real-Time Queue Tracking

## Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

AI:
- Ollama
- Qwen3:4B

Email Service:
- Brevo SMTP

## Setup

Backend:

npm install
npm run dev

Frontend:

npm install
npm run dev
